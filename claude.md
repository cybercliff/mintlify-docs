# MyChaiClub Documentation

This is a Mintlify documentation site for MyChaiClub, a platform for managing recurring donation campaigns.

## Project Structure

- `docs.json` - Mintlify configuration (navigation, theme, colors)
- `*.mdx` - Documentation pages written in MDX format
- `logo/` - Logo assets

## Documentation Sections

### Organization
- **Getting Started** - Quickstart guide for campaign setup
- **Campaign Setup** - Content, payment tiers, integrations, embedding
- **Campaign Administration** - Member management, payments, emails

### Donor
- Coming soon features for donor self-service

### General
- Support and contact information

## Development

Run the Mintlify development server:

```bash
npx mintlify dev
```

## Deployment

Documentation is automatically deployed when changes are pushed to the main branch.
