---
title: 'Embed Campaign Page'
description: 'Step-by-step instructions for embedding the campaign page on your existing website'
---

The campaign page can be embedded on almost any website. Contact support if you need additional assistance.

## Step 1: Get Your Embed Code

1. Login to [mychaiclub.com](https://mychaiclub.com) and navigate to **Promote** in the menu
2. Click **Embed Campaign**
3. Copy the embed code

## Step 2: Add to Your Website

Paste the embed code on your website as HTML. See platform-specific instructions below.

<AccordionGroup>
  <Accordion title="Squarespace" icon="square">
    Follow the [Squarespace embed block instructions](https://support.squarespace.com/hc/en-us/articles/206543617-Embed-blocks).

    <Note>
      Select **embed as code snippet** (not URL) when pasting. The campaign will automatically adjust to the correct size.
    </Note>
  </Accordion>

  <Accordion title="Wix" icon="wix">
    Follow the [Wix embedding instructions](https://support.wix.com/en/article/wix-editor-embedding-a-site-or-a-widget).

    <Note>
      Select **embed as code** (not URL) when pasting. The campaign will automatically adjust to the correct size.
    </Note>
  </Accordion>

  <Accordion title="ChabadOne" icon="globe">
    #### Create the Embed Page

    1. Login to [chabadone.org](https://chabadone.org)
    2. Click **Sites**
    3. Click **Publishing**
    4. Under "What would you like to create", select **New Web Document**
    5. Select the page location and click **Create**
    6. Enter a page name
    7. In the textbox, click **Source** to enable code mode
    8. Paste your embed code
    9. Click **Save**
    10. Click **Advanced Options** in the menu above the textbox
    11. Under **Page Width Setting**, select **Expand Body to full width of page**
    12. Click **Save**

    #### Add Link to Your Site

    Create a link to the newly created page in a prominent location on your website.
  </Accordion>
</AccordionGroup>
